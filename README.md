# Jarir Bookstore Management Frontend

A modern and responsive bookstore management system frontend built with React, Tailwind CSS, Redux Toolkit, and shadcn/ui. This application provides an intuitive interface for managing books, inventory, orders, customers, and reports.

## 🚀 Features

- 📚 Book Management

  - Add, edit, delete, and view books
  - Book categorization
  - Search and filtering

- 📦 Inventory Management

  - Stock tracking
  - Inventory updates
  - Low-stock alerts

- 👥 Customer Management

  - Customer profiles
  - Purchase history
  - Customer information management

- 🛒 Order Management

  - Create and manage orders
  - Order status tracking
  - Order history

- 📊 Dashboard & Analytics

  - Sales overview
  - Inventory reports
  - Business insights

- 🔐 Authentication & Authorization

  - Secure login system
  - Role-based access control
  - Protected routes

- 📱 Fully Responsive Design
  - Mobile-friendly interface
  - Tablet support
  - Desktop optimization

## ⚙️ Installation

### Clone Repository

git clone https://github.com/your-username/jarir-bookstore-management-frontend.git

### Navigate to Project

cd jarir-bookstore-management-frontend

### Install Dependencies

bun install

### Start Development Server

bun dev

## 🔑 Environment Variables

Create a .env file in the root directory and add:

VITE_API_URL=http://localhost:5000/api

## 📦 Build Project

npm run build

## 🧪 Run Preview

npm run preview
